# 🐇 Taverna Toca do Coelho

Sistema desenvolvido em PHP com foco no gerenciamento e armazenamento de dados utilizando banco MySQL.  
Este projeto representa um MVP (Minimum Viable Product) criado como base inicial para o Projeto Conselho.

---

## 🎯 Objetivo

Fornecer um sistema simples e funcional para controle de uma taverna temática, incluindo gestão de clientes, comandas, pedidos, jogos e estoque.

---

## ⚙️ Funcionalidades (em desenvolvimento)

- Cadastro de clientes  
- Controle de comandas  
- Registro de pedidos (itens, pratos e bebidas)  
- Gerenciamento de jogos disponíveis  
- Controle de ingredientes e estoque  
- Associação de itens às comandas  

---

## 🛠️ Tecnologias Utilizadas

- PHP  
- MySQL  
- HTML  
- CSS  
- JavaScript  

---

## 🧱 Estrutura do Projeto

O sistema é baseado em uma estrutura modular, com foco na organização do banco de dados e nas relações entre entidades como:

- Cliente  
- Comanda  
- Itens  
- Jogos  
- Pratos e Bebidas  
- Ingredientes  

---

## 🚧 Status do Projeto

Em desenvolvimento (versão inicial / MVP).  
Futuramente será integrado e expandido como parte do Projeto Conselho.
